Quick Conversion V1.0

Python GUI program that allows the user to quickly convert through common sequence file formats. 

Supported formats are as follows:
FASTA
Clustal
Stockholm
Fastq (fastq-sanger)
Pir
Nexus
tab

1) Select file you wish to convert using the "Select File to Convert" button
2) Select the input format
3) Select the output format
4) Press Convert - The output file will have the same name as the input, but with the new extension and file format
